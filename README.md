# Random Cat Viewer

A polished random cat gallery that pulls fresh feline profiles from the FreeAPI catalog and lets you page through your growing history without losing past finds.

## Live Demo

**[projects.ashish.pro/cats](https://projects.ashish.pro/cats)**

## Features

- Fetches random cat data from the FreeAPI endpoint defined in .env
- History-based pagination with Previous, Next, and New Cat controls
- Rich cat profile cards with temperament, origin, and stats
- Responsive layout with Tailwind-powered styling and Lucide icons

## Tech Stack

- React + Vite
- Tailwind CSS
- Lucide React
- FreeAPI Random Cat endpoint
